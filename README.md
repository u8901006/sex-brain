# Sex Brain · 性學文獻日報

每日自動從 PubMed 抓取性學、性諮商、性神經科學領域最新文獻，由 AI 分析彙整後生成日報。

## 涵蓋領域

- 核心性學 / 性醫學期刊
- 性諮商 / 心理性治療期刊
- 性神經科學 / 神經內分泌期刊

## 技術架構

- **資料來源**: PubMed E-utilities API
- **AI 分析**: Zhipu GLM-5.1
- **部署**: GitHub Actions + GitHub Pages
- **排程**: 每日台北時間 14:00
